일반 정보
--------------------

• From: 유저 에이전트의 이메일 정보

• Referer: 이전 웹 페이지 주소

• User-Agent: 유저 에이전트 애플리케이션 정보

• Server: 요청을 처리하는 오리진 서버의 소프트웨어 정보

• Date: 메시지가 생성된 날짜

  -어떤 종류의 브라우저에서 장애가 발생하는지 파악 가능
